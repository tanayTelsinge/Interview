# Graph Algorithms

This directory contains implementations and explanations of various graph algorithms.

## Directory Structure

### Topological Sort (`/topological_sort`)
- `ReactTopologicalSort.java` - Implementation of topological sort for package dependencies
- `TopologicalSortBFS.java` - BFS-based implementation of topological sort
- `TopologicalSortDFS.java` - DFS-based implementation of topological sort
- `CourseSchedule2.java` - LeetCode course schedule problem using topological sort
- `topological_sort.md` - Detailed explanation and examples of topological sort
