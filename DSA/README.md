# Data Structures and Algorithms

This repository contains implementations and explanations of various data structures and algorithms topics.

## Directory Structure

### Arrays (`/arrays`)
- Array manipulation problems and techniques
- Common array patterns and solutions

### Dynamic Programming (`/DP`)
- Dynamic Programming concepts and problems
- Solutions with memoization and tabulation

### Graph Algorithms (`/graph`)
- Graph traversal algorithms
- Topological Sort
- Path finding algorithms
- Network flow problems

### Mathematics (`/mathematics`)
- Mathematical algorithms and concepts
- Number theory
- Combinatorics

### Problems (`/problems`)
- Collection of specific problem solutions
- Interview questions

### Recursion (`/recursion`)
- Recursive algorithms
- Backtracking problems

### Searching (`/searching`)
- Binary Search
- Complete Search techniques
- Search algorithms

### Sliding Window (`/sliding_window`)
- Sliding window technique
- Problems and patterns

## Problem Categories
1. Array Manipulation
